2-Back
This task compares the position of letters displayed on the screen.
Please compare the position of the letter displayed on the screen to the position of the letter presented two letters (trials) previously, i.e. is the position of the displayed letter in the same position as two letters prior.
For example:
Compare the position of the 3rd letter to the position of the 1st letter, the position of the 4th letter to the position of the 2nd letter, the position of the 5th letter to the position of the 3rd letter, the position of the 6th letter to the position of the 4th letter and so on...
If the position of the letter matches the position of the letter presented two trials previously, please press M on the keyboard, for Match, as quickly as possible.
If the position of the letter does not match the position of the letter presented two trials previously, please press N on the keyboard, for No Match, as quickly as possible.
To begin, press the START button (A green dot will warn you that a letter is about to appear)
This task starts IMMEDIATELY after pressing START
